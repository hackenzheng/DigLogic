    课后作业

    每个 always 块被触发时，reg变量的赋值计算用的是所涉及到的变量触发时刻的值，但是否受同时触发的其他 always 计算结果影响？

    以下代码有三个always块，分别对 a，b，c进行赋值，当 a = 2时将b置为3，当b=3时将c置为4。那么当时钟上升沿时第一个always块将 a 置为了2，此时 b，c会立即同步置为3和4吗？

    请自行编写testbench对以下代码进行仿真，并结合波形分析说明做出回答。

    module test(
    input wire clk,
    input wire rst,
    output [7:0] data_out
);

    reg [7:0] a;
    reg [7:0] b;
    reg [7:0] c;

    assign data_out=c;

    always @(posedge clk)
    begin
        if(rst)
            a <= 8'd0;
        else
            a <= 8'd2;
    end

    always@(posedge clk)
    begin
        if(rst)
            b <= 8'd0;
        else if(a == 8'd2)
            b <= 8'd3;
    end

    always@(posedge clk)
    begin
        if(rst)
            c <= 8'd0;
        else if(b == 8'd3)
            c <= 8'd4;
    end

endmodule
